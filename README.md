#GMAO HVAC en Python

Proyecto de gestión de mantenimiento para sistemas de refrigeración.

## Funcionalidades

- Registro de equipos HVAC
- Cálculo de sobrecalentamiento y subenfriamiento
- Diagnóstico automático
- Seguimiento de rendimiento (COP)
- Detección de degradación
- Exportación de datos (JSON/CSV)

## Técnologías

- Python
- CoolProp
- Json /CSV

## Objetivo

Aplicar programación a mantenimiento HVAC para análisis y diagnóstico de sistemas frigorificos.

## Ejemplo de uso
1. Añadir equipo
2. Registrar medición (SH, SC, COP)
3. Ver diagnóstico
